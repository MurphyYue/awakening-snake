## Installation

1. Clone the repository
   ```bash
   git clone https://github.com/MurphyYue/awakening-snake.git
   ```
2. Install dependencies
   ```bash
   npm install
   ```
3. Start the development server
   ```bash
   npm run dev
   ```

## Controls

- **Arrow Keys**: Change snake direction
- **Spacebar**: Pause/Resume game
